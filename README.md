# Network Intrusion Detection System

A web-based Network Intrusion Detection System (IDS) developed using Java, Spring Boot, MySQL, Thymeleaf, and Chart.js.

## Features
- Real-time traffic monitoring
- Attack classification (DDoS, Port Scan, SQL Injection, Brute Force)
- Severity analysis
- Attack alerts
- Interactive dashboard
- Traffic visualization using charts
- CSV report export

## Technologies Used
- Java
- Spring Boot
- MySQL
- JPA/Hibernate
- Thymeleaf
- HTML
- CSS
- JavaScript
- Chart.js
